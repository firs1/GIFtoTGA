GifToTGA — это инструмент для конвертации анимационных GIF файлов в формат TGA с объединением всех фреймов GIF в одно изображение. Это особенно полезно для использования Warcraft, где необходимо конвертировать анимации в виде текстур.

Основные возможности:
1. Загружает GIF файл и извлекает все его фреймы.
2. Преобразует каждый фрейм GIF в формат TGA с возможностью настройки размера фрейма.
3. Собирает все фреймы в одном TGA файле, который можно использовать в играх или других приложениях.
4. Поддерживает возможность выбора размера фреймов для конечного TGA изображения.

Как использовать:
1. Скачать архив giftotga, разорхивировать и запустить giftoTGA.exe
2. Выберите исходный GIF файл.
3. Установите размеры фреймов (по умолчанию используются размеры первого кадра).
4. Конвертируйте файл в TGA, который будет готов для использования в вашем проекте.

  Далее информация для тех, кто будет использовать ее для создания ВА.
  Первым делом необходимо создать .tga файл данной программой, после необходимо сделать .blp файл. Чтобы сделать такой файл, можно воспользоваться данной утилитой: https://xgm.guru/p/wc3/blplab . Имейте ввиду, при больших размерах .tga файла blp lab может работать очень долго и не отвечать, но необходимо подождать, поэтому рекомендую выставлять размер фрейма 128x128, а лучше 64x64. А также сохранить файл в формате BLP2.
  Создание ВА. Первым делом необходимо закинуть получившийся .blp файл по пути: "Interface\AddOns\WeakAuras\Media\Textures\". Далее необходимо сделать новую индикацию — Стоп-кадр. Ввести кол-во строк и столбцов из .tga файла. Просмотреть кол-во строк и столбцов можно использовав бесплатную утилиту paint.net - https://paintnet.ru/download/ . Ввести кол-во фреймов пишет программа giftoTGA. Ширина и высота файла также пишется после завершения работы программы. Ширина и высота кадры - то что выставили вручную, либо то исходных размер GIF. Частоту смены кадров, конец анимации с его началом выставляется по вкусу. Примеры ВА, а также исходные .blp файлы также прикреплены к релизу.
