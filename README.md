GifToTGA — это инструмент для конвертации анимационных GIF файлов в формат TGA с объединением всех фреймов GIF в одно изображение. Это особенно полезно для использования Warcraft, где необходимо конвертировать анимации в виде текстур.

Основные возможности:
1. Загружает GIF файл и извлекает все его фреймы.
2. Преобразует каждый фрейм GIF в формат TGA с возможностью настройки размера фрейма.
3. Собирает все фреймы в одном TGA файле, который можно использовать в играх или других приложениях.
4. Поддерживает возможность выбора размера фреймов для конечного TGA изображения.

Как использовать:
1. Скачать архив giftotga, разорхивировать и запустить giftoTGA.exe
2. Выберите исходный GIF файл.
3. Установите размеры фреймов (по умолчанию используются размеры первого кадра).
4. Конвертируйте файл в TGA, который будет готов для использования в вашем проекте.

Инструкция по созданию WeakAura (ВА)
1. Создание .tga файла:
Для начала используйте программу для конвертации GIF в формат .tga. После этого получите файл .tga, который будет использоваться для создания анимации.
2. Создание .blp файла:
Чтобы преобразовать .tga файл в формат .blp, воспользуйтесь утилитой BLP Lab https://xgm.guru/p/wc3/blplab.
Учтите, что при работе с большими .tga файлами BLP Lab может работать довольно долго или не отвечать. Это нормально, нужно подождать, пока процесс не завершится.
Рекомендуется выставить размеры фреймов 128x128 или 64x64, чтобы ускорить процесс. После этого сохраните файл в формате BLP2.
3. Загрузка .blp файла в WeakAura:
Переместите полученный .blp файл в директорию Interface\AddOns\WeakAuras\Media\Textures\.
4.Создайте новую индикацию типа Стоп-кадр.
5.Введите количество строк и столбцов из вашего .tga файла. Эти параметры можно проверить с помощью программы Paint.NET https://paintnet.ru/download/ .
6.Количество фреймов можно узнать после завершения работы программы giftoTGA. Также будут указаны ширина и высота файла, которые можно использовать для настройки.
7.Ширина и высота фреймов могут быть как исходными размерами GIF, так и теми, которые вы указали вручную.
8. Настройка анимации:
Частоту смены кадров и переходы между кадрами можно настроить по вашему усмотрению.
